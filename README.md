# 单词打地鼠 (Word Whack-A-Mole)

一款帮助小学生学习英语单词的趣味益智游戏！

## 游戏特色

- 🎮 有趣的打地鼠玩法
- 📚 人教版新起点英语词汇
- ⭐ 分数系统和连击奖励
- 🏆 多级称号系统
- 🎨 精美可爱的界面设计
- 🔊 互动音效反馈

## 如何运行

### 首次运行

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中打开显示的地址（通常是 http://localhost:8088）

### 后续运行

直接运行：
```bash
npm run dev
```

## 游戏玩法

1. **选择范围**：从首页选择学习范围（1年级上册、下册，2年级上册等）
2. **开始游戏**：游戏界面上方会显示一个单词
3. **点击地鼠**：点击头顶显示正确答案的地鼠
4. **获得分数**：
   - 正确答案：+10分（连击有额外加分）
   - 错误答案：-5分
5. **时间限制**：60秒内尽可能多得分

## 技术栈

- React 18
- Vite
- CSS3 动画
- 响应式设计

## 适合人群

6-12岁小学生，特别适合正在学习人教版新起点英语的学生。
